# Список изменений

В этом файле описываются заметные для пользователей изменения pb-ftp.

## [1.0.4] - 2026-06-20

### Исправлено

- Локальный endpoint обновления теперь совместим с будущими полями метаданных Android-клиента.
- Перед перезапуском после self-update запускается сканирование накопителя PocketBook, чтобы список приложений надёжнее видел заменённый launcher.
- Перед выходом для перезапуска pb-ftp даёт ответу об обновлении успеть дойти до Android-клиента.

## [1.0.3] - 2026-06-20

### Добавлено

- Добавлена публикация локализованного списка изменений для окна обновления сервера PocketBook в Android-приложении.

### Исправлено

- Добавлено автоматическое восстановление встроенного FTP-сервера после пробуждения PocketBook, если процесс сервера завершился во сне.
- Добавлено штатное FTP QUIT-завершение перед остановкой встроенного FTP-сервера при выходе из приложения.
